Endpoints para registrar eventos analíticos y consultar analíticas privadas de lectura.

Flujo recomendado para investigar un caso:
1. Usa **Obtener analítica del proyecto** para detectar una métrica o un rango sospechoso.
2. Usa **Listar conversaciones del proyecto** para localizar las conversaciones relevantes dentro de ese rango.
3. Usa **Obtener detalle de una conversación** para inspeccionar transcript, eventos, feedback y metadata.
