Endpoints para registrar eventos analíticos y consultar cómo está funcionando el asistente.

Flujo recomendado para revisar resultados:
1. Usa **Obtener analítica del proyecto** para ver tendencias, uso y satisfacción.
2. Usa **Listar conversaciones del proyecto** para entender qué están preguntando los visitantes.
3. Usa **Obtener detalle de una conversación** para revisar un caso concreto cuando necesites más contexto.
