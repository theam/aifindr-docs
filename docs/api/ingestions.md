## Start Ingestion

```bash
POST /admin/api/projects/{projectId}/ingest
{
  "source": "gcs://bucket/path",
  "type": "gcs"
}
```

## Check Status

```bash
GET /admin/api/projects/{projectId}/ingestions
```
