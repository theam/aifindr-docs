# **Cómo se integra?**

AI Findr se integra en tu web o producto de **dos maneras**:

## **1. Widget Web**

Un componente listo para usar que se integra con un simple script.
Se personaliza desde el Hub con tu tipografía, colores y logo, y se adapta automáticamente a tu diseño.

## **2. API de AI Findr**

Conecta directamente con la API y construye tu propia interfaz.
Tú diseñas la experiencia visual; AI Findr gestiona el entendimiento y las respuestas.
