Endpoints for analytics event tracking and private read access.

Recommended flow to investigate a case:
1. Use **Get Project Analytics** to identify a suspicious metric or date range.
2. Use **List Project Conversations** to find the relevant conversations in that range.
3. Use **Get Conversation Detail** to inspect transcript, events, feedback, and metadata.
