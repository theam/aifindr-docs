Endpoints to track analytics events and review how the assistant is performing.

Recommended flow to review results:
1. Use **Get Project Analytics** to see trends, usage, and satisfaction.
2. Use **List Project Conversations** to understand what visitors are asking.
3. Use **Get Conversation Detail** to review one specific case when you need more context.
