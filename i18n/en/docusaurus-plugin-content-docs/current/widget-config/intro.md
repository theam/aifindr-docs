# **AIFindr** Widget

The **AIFindr** widget integrates a conversational assistant into your website that **answers questions, guides users, and increases conversion** without interrupting navigation.

## Key Benefits

| Benefit                        | Detail                                                                    |
| ------------------------------ | ------------------------------------------------------------------------- |
| **Ultra-fast installation**    | Copy-paste a single line and you're done.                                 |
| **Fully responsive**           | Adapts to mobile, tablet, and desktop.                                    |
| **Respects your *navbar***     | Never covers menus or headers.                                            |
| **Highly customizable**        | Colors, texts, and behavior are configured with simple options.           |
| **Real-time context**          | Personalizes responses based on your funnel state or user.                |
